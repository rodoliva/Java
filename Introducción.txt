Construir una aplicación que implemente y permita comparar el desempeño de cuatro métodos de ordenamiento: Inserción, Selección, Burbuja y QuickSort, aplicados todos sobre un arreglos de números enteros.
La idea de esta aplicación es que pueda generar experimentos con cada uno de estos métodos con distintos tamaños de largos para el arreglo a construir, que va desde un largo inicial de 500 posiciones hasta uno de 20000 posiciones con un intervalo de 500 posiciones generándose 40 experimentos con largos distintos.
Los datos que se generan para el arreglo se obtienen a través del método Math.random() en la clase Tarea que genera números aleatorios desde 0 hasta el 999.Este método entrega números decimales (double) por lo que en la clase Tarea se multiplica por 1000 el método y luego se realiza un casting para convertirlo a un número entero (int).
Para cada uno de los tamaños del arreglo, se realizaron 20 arreglos con datos distintos determinando el tiempo promedio de desempeño para cada método de ordenamiento medido en milisegundos. Además, para cada método de ordenamiento se midió el tiempo de ejecución  para un mismo arreglo con cada uno de sus largos, con los datos ordenados ascendentemente y descendentemente para comparar posteriormente su desempeño.
